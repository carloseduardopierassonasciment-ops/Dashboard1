# Dashboard1

Este projeto é um **dashboard web simples**, desenvolvido utilizando **HTML, CSS e JavaScript**, com o objetivo de exibir informações de forma visual e organizada diretamente no navegador.

O foco do projeto é praticar conceitos de **frontend**, como estruturação de páginas, estilização e manipulação de dados com JavaScript.

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura da aplicação  
- **CSS3** – Estilização e layout  
- **JavaScript (Vanilla)** – Lógica e interatividade
